手动解析下载腾讯视频

Aug 18, 2020 21:09 · 589 words · 2 minute read 下载 腾讯视频 视频解析

  • 视频解析
  • FDM下载
  • FFmpeg合并视频
  • FFmpeg提取音频

妈咪想我帮她下载某微信公众号里大悲咒的音频。

视频解析

为此，我需要先获得原视频链接，我用电脑版登陆微信，右键选择用默认浏览器打开网页。 我用Chrome浏览器打开网页后，我右键页面，审查网页元素，来到Network选项，过滤器设置为media，侦听来自网页的音视频类型的文件。原视频是29分钟的，我发现视频被网站拆分为了几段视频传输。这意味着需要把所有视频下载下来，然后合并为一个视频。

FDM下载

于是我用两倍速度，让视频跑一遍，然后得出了所有分段视频的地址。于是我右键在网页用FDM下载所有视频，然后共有5段视频被下载下来了。

FFmpeg合并视频

视频下载下来后，我用ffmpeg把这几段视频合并为一个视频。如果用传统视频编缉软件，实在太慢了。所以我用ffmpeg敲几个命令就搞定。此次合并视频的ffmpeg命令是： ffmpeg -f concat -safe 0 filelist.txt -c copy output.mp4 在这里filelist.txt里写着要合并的视频文件列表，格式类似于

file 1.mp4

file 2.mp4

file 3.mp4

FFmpeg提取音频

上面一步在几秒时间内就把五段视频合并生成了一个output.mp4文件，多方便呐！接下来呢，就是提取音频，也是敲几行命令。 ffmpeg -i output.mp4 -nv output.mp3 这样-nv意思是not video的意思，就是忽略视频。这样就成功从视频里提取出了一个mp3音频文件。

那么到这里任务就完成啦！我把这个音频文件，放到u盘里，插到收音机里，就可以播放了。(收音机是我母亲的佛友送的，感谢哈！
