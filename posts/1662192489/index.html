电脑ssh连接termux

Sep 3, 2022 16:08 · 88 words · 1 minute read termux 手机

  • 电脑安装open ssh
apt install openssh
  • 设置密码
passwd
  • 生成keygen
ssh-keygen -t rsa
  • 把*.pub公钥复制到手机的.ssh目录下
cat ./*.pub >> authorized_keys
  • 手机运行得到用户名
whoami
  • 电脑连接
ssh 用户名@192.168.123.81 -p 8022
